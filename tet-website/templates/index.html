Framework Tools

Discover the integrated solutions that make up the Transparency Assessment Framework.

Browser Extension

A browser extension that guides developers through tasks, collects interaction data, and captures UX feedback to assess portal transparency.

Dashboard

Delivers powerful insights into analyzed portals with KPIs and a comprehensive transparency report based on established guidelines.

UX-Tracking

A tool that collects real-time user interaction data, maps user journeys, identifies key touchpoints, and delivers valuable UX insights for both web and mobile environments.

How Does a Portal Transparency Evaluation Work?

Learn how to use our tools to assess the transparency of your software ecosystem portal.

1. Create Evaluation

After creating an account, you can start a new assessment for your portal by selecting the specific processes to be tested by developers. All you need to do is provide the link to your portal.

2. Data Collection

When an assessment is created, a code will be generated so developers can join it through the browser extension. The extension will record their interactions with the portal, collecting relevant information for the evaluation.

3. Data Analysis

Access the dashboard to explore the collected data, generate detailed reports, and view performance charts. Use these insights to understand usage patterns and identify opportunities to improve your portal’s transparency.

4. Correction

Review the generated report, go over the recommended transparency guidelines, and implement the necessary corrective actions to enhance your portal based on the identified issues.
