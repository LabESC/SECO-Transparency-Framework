Como Funciona Uma Avaliação de Transparência em Portais?

Aprenda a utilizar nossas ferramentas para avaliar a transparência do seu portal de ecossistema de software.

1. Criar avaliação

Após criar uma conta, é possível iniciar uma nova avaliação para o seu portal, selecionando os processos específicos que devem ser testados pelos desenvolvedores. Para isso, basta fornecer o link para o portal.

2. Coleta de Dados

Ao criar uma avaliação, um código será gerado para que os desenvolvedores possam participar dela por meio da extensão de navegador. Ela irá registrar as interações dos desenvolvedores com o portal, coletando informações relevantes para a avaliação.

3. Análise dos dados

Acesse o dashboard para explorar os dados coletados, gerar relatórios detalhados e visualizar gráficos de desempenho. Utilize esses insights para entender padrões de uso e identificar pontos de melhoria na transparência do seu portal.

4. Correção

Consulte o relatório gerado, revise as diretrizes de transparência recomendadas e implemente as ações corretivas necessárias para aprimorar seu portal com base nos pontos identificados.
